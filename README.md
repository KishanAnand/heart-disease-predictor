# heart-disease-predictor

A machine learning web application that predicts the likelihood of heart disease based on patient medical data. Built using Streamlit and multiple classification models.

Predicts heart disease risk using multiple ML models:
 - Logistic Regression
 - Naive Bayes
 - Support Vector Machine (SVM)
 - K-Nearest Neighbors (KNN)
 - Decision Tree
 - XGBoost
 - Interactive web interface using Streamlit
 -  Real-time predictions based on user input
 -   Displays risk level and probability (if supported by model)

Tech Stack
  - Python
  - Streamlit
  -  NumPy
  -  Scikit-learn
  -   XGBoost
  -   Joblib

RUN THE APP
  - streamlit run app.py
